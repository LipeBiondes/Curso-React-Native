- Instalar Expo cli

npm install -g expo-cli​​

- Criar um novo projeto usando Expo:

expo init nomeApp​​​

- Rodar o servidor do Expo:

> Usando expo mesmo:​

expo start

> Usando yarn:

yarn start

> Usando npm:

npm start
