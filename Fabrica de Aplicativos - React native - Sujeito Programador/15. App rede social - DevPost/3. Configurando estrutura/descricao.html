Vamos aos passos para instalar:

1) Primeiro:

npm install @react-navigation/native@^5.x​


​2) Agora instalando restante das bibliotecas pedidas:

npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view


3) Instalar navegação em Stack:

npm install @react-navigation/stack@^5.x​


3) Instalar navegação em Tab:

npm install @react-navigation/bottom-tabs@^5.x​

** Se você está no android então pode rodar seu projeto normalmente e seguir como fazemos na aula **

>> Se estiver no Mac com iOS então siga os passos abaixo:

- Volte ao cmd e digite: cd ios para acessar a pasta iOS e agora digite no cmd: pod install

Após ele baixar tudo do pod install pode voltar a pasta do projeto digitando no cmd cd ..

E agora já pode rodar normalmente seu projeto.
